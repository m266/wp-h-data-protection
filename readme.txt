== Description ==
Datenschutz-Plugin

== Installation ==
1. Upload 'beispiel-plugin-with-submenu'  to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin in Dashboard -> WP H-Data Protection
